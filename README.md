DotFeather is a lightweight generic 2D gameengine for C#/.NET Standard 2.0.

## To Build

```
git clone https://github.com/xeltica/DotFeather.git
cd DotFeather
nuget restore
dotnet build
```

I'm going to publish DotFeather to nuget ASAP.
